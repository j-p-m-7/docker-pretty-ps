# docker-pretty-ps

### Description

Simple command line tool to output docker containers in a pretty format


### Installation

Can be installed with uv via:

```bash
uv tool install git+https://github.com/j-p-m-7/docker-pretty-ps
```


### Sample usage

```bash
$ docker-pretty-ps
```
